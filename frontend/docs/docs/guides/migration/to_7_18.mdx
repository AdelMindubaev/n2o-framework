### Изменения в XML API {#changes-xml-api}

- Произошли небольшие изменения в механизме маппинга java провайдера.
К возможности осуществлять маппинг по индексу аргумента (`[0]`, `[1].name`)
добавлена также возможность делать это по его имени (`['id']`, `['person'].name`).
Также пропала возможность автоматического маппинга внутренних полей при указании только одного аргумента.
Если раньше, чтобы смаппить поле id в критерий можно было указать маппинг `id`.
То теперь во избежании неоднозначности нужно указывать индекс или имя аргумента (например: `[0].id` или `['criteria'].id`).
