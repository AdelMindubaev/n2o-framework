### Изменения в XML API {#changes-xml-api}

- У ячеек `image` атрибут `width` изменил тип с числа на строку, если вы задавали этот атрибут ранее, необходимо добавить
в конце `px`. Например, было `width="50"` должно стать `width="50px"`

- У полей `image` появился атрибут `action-id` для ссылки на действие, которое необходимо выполнить при клике на картинку.

### Изменения в Java API

- Класс `ValidateProcessor` переименован в `SourceProcessor`
- Метод `SourceTransformer.transform(S source)` удален, вместо него используйте метод `SourceTransformer.transform(S source, SourceProcessor p)`
- Изменено api автотестов. Метод для проверки имени первой страницы в бредкрамбе переименован с `parentTitleShouldHaveText`
на `firstTitleShouldHaveText`.
