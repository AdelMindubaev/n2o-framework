### Изменения в XML API {#changes-xml-api}

- Из поля `<alert>` удалены атрибуты `fade`, `tag` и `footer`, атрибут `header` переименован в `title`. Текст для
отображения в алерте теперь задается в новом атрибуте `text`. Также был добавлен атрибут `close-button`, принимающий
значения `true/false` и отвечающий за наличие кнопки закрытия алерта (по умолчанию `false`), атрибут `href`, в котором
можно указать ссылку или относительный путь приложения для перехода при нажатии на алерт, атрибуты `style` и `class`,
позволяющие добавить стиль и CSS класс алерту.
- Добавлено новое действие `<alert>`, для которого помимо свойств поля `<alert>` можно задать место и длительность
отображения в атрибутах `placement` (по умолчанию `top`) и `timeout` (по умолчанию `3000`).
