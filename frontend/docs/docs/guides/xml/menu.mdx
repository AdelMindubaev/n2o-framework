### &lt;menu-item&gt; {#menu-item}

Компонент меню

**Атрибуты**

|Наименование|Тип      |Описание                     |Значение по умолчанию|
|------------|---------|-----------------------------|---------------------|
|id          |Строка   |Идентификатор компонента меню|                     |
|name        |Строка   |Наименование компонента меню |                     |
|icon        |Строка   |Иконка компонента меню       |                     |
|badge       |Число    |Бадж компонента меню         |                     |
|badge-color |Строка   |Цвет баджа                   |                     |

**Тело**

Действие открытия страницы [Открытие вложенной страницы](../xml/action#open-page), [открыть страницу браузера](../xml/action#a)

**Пример**

```xml
    <menu-item id="notif"
               name="Уведомления"
               icon="fa fa-bell"
               badge="2"
               badge-color="warning"/>
```

### &lt;dropdown-menu&gt; {#dropdown-menu}

Выпадающее меню

**Атрибуты**

|Наименование|Тип                    |Описание                      |Значение по умолчанию|
|------------|-----------------------|------------------------------|---------------------|
|id          |Строка                 |Идентификатор выпадающего меню|                     |
|name        |Строка                 |Наименование выпадающего меню |                     |
|image       |Строка                 |Путь к изображению            |                     |
|image-shape |square, circle, rounded|Форма изображения             |rounded              |

**Тело**

[Компонент меню](../xml/menu#menu-item), разделитель компонентов выпадающего меню

**Пример**

```xml
<dropdown-menu id="user"
               name="Виктория"
               image="/static/users/vika91.png"
               image-shape="circle">
    <menu-item>...</menu-item>
    <devider/>
</dropdown-menu>
```

### &lt;divider&gt; {#divider}

Разделитель компонентов выпадающего меню

**Пример**

```xml
<devider/>
```
